Beat That!

A dice game

CPU's dice

Player's dice

  1. The objective of the game is to roll and create the highest number.
  2. Each dice is considered its own entity.
  3. For example: if player rolls a 4 and a 6, the highest number they can create 64.
  4. The player that rolls and creates the highest number wins the round. The game ends after 3 rounds.
